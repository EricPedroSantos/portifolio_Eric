CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a apresentação e o layout de documentos HTML e XML. É uma tecnologia fundamental da web que permite que os desenvolvedores criem páginas web atraentes e visualmente agradáveis.

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto, em português) e é uma linguagem de marcação usada para criar e estruturar o conteúdo de páginas web

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, que é amplamente utilizada na criação de conteúdo para a web. É uma das principais tecnologias da web moderna, juntamente com HTML e CSS.

Node.js é uma plataforma de desenvolvimento de software de código aberto construída em cima da engine V8 do Google Chrome. Ele permite que os desenvolvedores usem JavaScript tanto no lado do cliente quanto no lado do servidor.

React é uma biblioteca de JavaScript de código aberto, desenvolvida pelo Facebook, que é usada para construir interfaces de usuário (UI) para aplicativos web e móveis. Ele permite que os desenvolvedores criem componentes reutilizáveis que podem ser usados ​​para construir interfaces de usuário complexas e interativas

Vue.js é um framework de código aberto para construção de interfaces de usuário (UI) e aplicações web de uma única página (SPA). Ele foi criado por Evan You em 2014, e desde então se tornou uma das ferramentas mais populares para desenvolvimento front-end
